# Crossfit Streamlit

This is a dashboard to view crossfit data.

## Installation

Please have at least python3.10 on your computer.

1. Install [Pipenv](https://pypi.org/project/pipenv/) if not already installed via `pip install pipenv`

2. Install Virtual Environment and dependencies with `pipenv install`

3. Please download the following [dataset](https://www.kaggle.com/datasets/ulrikthygepedersen/crossfit-athletes?resource=download) and put the csv in the `temp/` directory.

## Running the development build

1. Run `pipenv run dev` in order to run in development mode
